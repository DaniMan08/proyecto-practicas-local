# 🎭 Proyecto Escuela de Teatro – Prácticas DAW  
Sistema interno de gestión con módulos de login, registro, mensajería (Email y WhatsApp) y panel de administración.

---

## 🚀 Funcionalidades principales

- Inicio de sesión con roles (Admin / Empleado)
- Gestión de usuarios, alumnos, profesores y grupos
- Envío de mensajes por:
  - ✉️ Email (PHPMailer)
  - 💬 WhatsApp Web
- Historial de mensajes enviados
- Inserción automática en base de datos:
  - mensaje
  - mensaje_destinatario
- Control de permisos y sesiones
- Estructura organizada por carpetas (public / config / src)

---

## 📦 Requisitos

- XAMPP (Apache + MySQL)
- Git instalado en el ordenador
- Visual Studio Code (recomendado)
- Navegador compatible con WhatsApp Web
